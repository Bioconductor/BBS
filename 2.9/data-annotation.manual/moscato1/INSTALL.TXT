Scripts in this dir are standalone scripts used to manually
build the "2.9-data-annotation" binary packages on moscato1.

Install on moscato1 from the 'biocbuild' account with:

  CD D:\biocbld
  svn co https://hedgehog.fhcrc.org/bioconductor/trunk/bioC/admin/build/BBS/2.9/data-annotation.manual/moscato1 bbs-2.9-data-annotation
  CD bbs-2.9-data-annotation
  MKDIR src-pkgs bin-pkgs Rlib

Then read README.TXT
