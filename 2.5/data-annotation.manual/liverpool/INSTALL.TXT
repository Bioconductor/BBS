Scripts in this dir are standalone scripts used to manually
build the "2.5-data-annotation" binary packages on liverpool.

Install on liverpool from the 'biocbuild' account with:

  CD E:\biocbld
  svn co https://hedgehog.fhcrc.org/bioconductor/trunk/bioC/admin/build/BBS/2.5/data-annotation/liverpool bbs-2.5-data-annotation
  CD bbs-2.5-data-annotation
  MKDIR src-pkgs bin-pkgs Rlib

Then read README.TXT
